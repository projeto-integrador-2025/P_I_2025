# 🤖 Chatbot com Flask + Groq API
Este projeto é um chatbot interativo desenvolvido com Python + Flask, que utiliza a API da Groq com o modelo LLaMA 3 para responder mensagens de forma inteligente.

A interface é simples e responsiva, e o backend faz chamadas diretas à API de linguagem natural para gerar respostas em tempo real.

### 🚀 Tecnologias Utilizadas
Python

Flask

OpenAI (Groq endpoint)

Flask-CORS para permitir requisições cross-origin

dotenv para gerenciamento de variáveis de ambiente

### 🛠️ Funcionalidades
Interface web com histórico de mensagens

Comunicação em tempo real com a API da Groq

Uso de variáveis de ambiente para segurança da chave da API

Armazenamento simples do histórico de conversa em memória
